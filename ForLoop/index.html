Introduction to Java

For loop:

Most programs involve doing something over and over again. To make things easier, the Java language has loops which can be used to repeat certain lines of code. In this lesson, we will learn about one particular type of loop, called the for-loop.

If you wanted to print out “Hello world"

Let’s say you wanted to print out onto the screen”Hello World” 10 times. How would you go about doing this? Well, for someone who doesn’t know about loops, they may simply type out 10 print statements. This will work, yes, but it is definitely the wrong way to go about it. The problem with this is that it will get very tedious when scaled up. What if you had to print something out 100, or even 1,000,000 times? Im sure no one wants to sit in front of a computer for hours typing out one million print statements. The correct way to do this is using a for-loop.

For-loops have a very specific structure: the header and the body. The body is where you write all the lines of code that you wish to be looped. The header, one the other hand, is divided into three sections. A rough outline of what a for-loop should look like is below:




It’s almost never a good idea to let a loop go on forever, so you will need to tell it:

  1. Where to start
  2. Where to end
  3. How to progress (increment)

This is where the header comes in. The first section of the loop will be used to initialize an int variable that will tell us at which number the loop begins. The middle section holds a condition, which as long as it’s true, the loop will continue, and when it is false the loop will terminate. The final section tells us how the int variable will change or progress.



Now here is a for-loop in action. We start with an int variable ‘i’ initialized to 1. The middle section says, “as long as the the variable ‘i’ is less than or equal to 5, go inside the body and execute any code in there”. The third section will add 1 to ‘i’ after executing the code in the body. So the order looks something like this:

  • ‘i’ = 1
  • Is i <= 5? Yes it is, so go into the body and print “Hello World”
  • Increment i, i = 2
  • Is i <= 5? Yes it is, so go into the body and print “Hello World”
  • Increment i, i = 3
  • Is i <= 5? Yes it is, so go into the body and print “Hello World”
  • Increment i, i = 4
  • Is i <= 5? Yes it is, so go into the body and print “Hello World”
  • Increment i, i = 5
  • Is i <= 5? Yes it is, so go into the body and print “Hello World”
  • Increment i, i = 6
  • Is i <= 5? No it is not. Exit loop and move on to any code after it.


The result is “Hello World” prints out 5 times. The beauty of this is that we can just as easily print it a 100 times by simply changing the condition in the for-loop to: i <= 100
