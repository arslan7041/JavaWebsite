Introduction to Java






Java is a programming language released by Sun Microsystems in 1995. In 2009, Oracle Corporation took over Java. Java is concurrent, class-based, object-oriented, and specifically designed to have as few implementation dependencies as possible. Java is used for various programming aspects including android development. It was designed to look like C++ but is simpler to use.
