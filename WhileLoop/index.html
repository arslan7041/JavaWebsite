While loop:

The second type of loop we will learn about is the while-loop. This loop starts with the keyword “while” and only has a condition in its header. The int variable must be initialized before it and incremented within the body itself, unlike the for-loop Here is a simple example:




We can also incorporate if-statements within the body of a loop. Let’s say we wanted to print all the numbers from 1 to 10, but then only the even numbers up to 20, in a single loop. We can use if-statements to do that.



Notice, within the loop, after printing, we use if-else clauses to either add 1 or 2 to ‘i’ depending on its value. Since we want to print 1 through 10, we only add 1 until ‘i’ is 9. Once ‘i’ hits 10, we only wish to print the even numbers, and thus we can now start adding 2 to it.

We can also use a boolean variable as a condition in the header of the loop.



Here we keep the loop running as long as ‘keep_looping’ is true. The variable ‘i’ starts at 1 and is incremented. Once it equals 10, we set the boolean variable to false, so the next time the condition is checked it will be evaluated as false and the loop will terminate. If we did not eventually set the boolean to false, the loop would go on forever.
