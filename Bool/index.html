Boolean Data Type:

The boolean data type is the simplest of them all, because it can hold only two values: true or false. You can think of a boolean variable like a switch, its either on (true) or off (false).

To declare a boolean variable, simply write the keyword boolean, followed by the name of the variable, and then you can assign it true or false. Note that true and false are keywords themselves, and not strings.




Notice that the “true” and “false’ are highlighted blue, indicating that they are keywords. The above code also demonstrates that, as with any other variable, you can change the value at anytime. Our variable “light_switch” was first initialized to “true” and then later changed to “false”. Printing out a boolean variable simply prints its true/false value.

Boolean’s can come in very handy as flags or signals that you can use to decide whether to do or not do something. We will demonstrate this in detail in a later lesson.
